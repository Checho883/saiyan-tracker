    1. From frontend/, install test dev dependencies:
       ```
       npm install -D vitest @testing-library/react @testing-library/jest-dom @testing-library/user-event jsdom @types/react @types/react-dom
       ```
       Note: @types/react and @types/react-dom may already be installed from the Vite scaffold — npm will handle deduplication.

    2. Add `"test": "vitest run"` script to `package.json` scripts section. If a `"test"` script already exists from the Vite scaffold, replace it.

    3. Create `vitest.config.ts`:
       ```typescript
       import { defineConfig } from 'vitest/config';
       import react from '@vitejs/plugin-react';

       export default defineConfig({
         plugins: [react()],
         test: {
           environment: 'jsdom',
           setupFiles: ['./src/test-setup.ts'],
           globals: true,
           css: true,
         },
       });
       ```

    4. Create `src/test-setup.ts`:
       ```typescript
       import '@testing-library/jest-dom/vitest';
       ```

    5. If `tsconfig.app.json` exists, ensure `src/test-setup.ts` and `src/__tests__/` are NOT excluded from TypeScript compilation. The test files need access to project types. If there's an `include` array, ensure it covers `src/**/*.ts` and `src/**/*.tsx`.
  </action>

  </verify>
