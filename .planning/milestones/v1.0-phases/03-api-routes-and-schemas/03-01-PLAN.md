    **Categories router (`backend/app/api/v1/categories.py`):**
    - `router = APIRouter(prefix="/categories", tags=["categories"])`
    - POST "/" — Create Category ORM object from CategoryCreate body, set user_id from dependency. db.add(), db.commit(), db.refresh(). Return CategoryResponse, status 201.
    - GET "/" — Query all categories for user. Return list[CategoryResponse].
    - GET "/{category_id}" — Query by id + user_id. HTTPException 404 if not found. Return CategoryResponse.
    - PUT "/{category_id}" — Query, update only non-None fields from CategoryUpdate using `model_dump(exclude_unset=True)`. db.commit(), db.refresh(). Return CategoryResponse.
    - DELETE "/{category_id}" — Query, db.delete(), db.commit(). Return 204 (no content). Use `Response(status_code=204)`.

    **Rewards router (`backend/app/api/v1/rewards.py`):**
    Same CRUD pattern as categories. Fields: title, rarity, is_active. List endpoint returns all rewards (no filter on is_active -- let frontend filter).

    **Wishes router (`backend/app/api/v1/wishes.py`):**
    Same CRUD pattern. Additional:
    - POST "/grant" — Accept WishGrantRequest body. Call `grant_wish(db, user, body.wish_id)`. Catch ValueError and raise HTTPException(400, detail=str(e)). db.commit(). Return WishGrantResponse.

    **Off-days router (`backend/app/api/v1/off_days.py`):**
    - `router = APIRouter(prefix="/off-days", tags=["off-days"])`
    - POST "/" — Accept OffDayCreate body. Check if already off day via is_off_day() -- if so, raise 409. Call mark_off_day(db, user.id, body.local_date, body.reason, body.notes). db.commit(). Return OffDayMarkResponse.
    - GET "/" — Query OffDay for user. Optional query param `month` (str, YYYY-MM format) to filter by month prefix. Return list[OffDayResponse].
    - DELETE "/{off_date}" — off_date is a path param (str, YYYY-MM-DD). Call cancel_off_day(db, user.id, off_date). If returns False, raise 404. db.commit(). Return 204.

    **Settings router (`backend/app/api/v1/settings.py`):**
    - `router = APIRouter(prefix="/settings", tags=["settings"])`
    - GET "/" — Return SettingsResponse from user object fields.
    - PUT "/" — Accept SettingsUpdate body. Update user fields from non-None values. Validate theme against VALID_THEMES (Literal type handles this via Pydantic). db.commit(), db.refresh(user). Return SettingsResponse.

    **Quotes router (`backend/app/api/v1/quotes.py`):**
    - `router = APIRouter(prefix="/quotes", tags=["quotes"])`
    - GET "/random" — Optional query param trigger_event (str|None). Query Quote table. If trigger_event provided, filter by it. Use `order_by(func.random()).first()` for SQLite random selection. If no quote found, raise 404. Build QuoteResponse with avatar_path computed as f"/avatars/{quote.character}.png". Return QuoteResponse.
