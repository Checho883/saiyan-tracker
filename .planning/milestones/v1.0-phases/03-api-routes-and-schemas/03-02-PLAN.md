    **Create `backend/app/api/v1/habits.py`:**
    `router = APIRouter(prefix="/habits", tags=["habits"])`

    **CRUD endpoints:**
    - POST "/" — Create Habit from HabitCreate body + user_id from dependency. db.add(), db.commit(), db.refresh(). Return HabitResponse, status 201.
    - GET "/" — Query Habit where user_id=user.id and is_active=True. Return list[HabitResponse].
    - GET "/{habit_id}" — Query by id + user_id. 404 if not found. Return HabitResponse.
    - PUT "/{habit_id}" — Query, update non-None fields from HabitUpdate via model_dump(exclude_unset=True). db.commit(), db.refresh(). Return HabitResponse.
    - DELETE "/{habit_id}" — Soft delete: set is_active=False. db.commit(). Return 204.

    **Check habit endpoint:**
    POST "/{habit_id}/check":
    1. Parse CheckHabitRequest body (local_date).
    2. Validate off day: `if is_off_day(db, user.id, body.local_date): raise HTTPException(409, "Cannot check habits on an off day")`
    3. Validate habit exists and belongs to user: query Habit by id + user_id. 404 if not found.
    4. Validate habit is due: `due_habits = get_habits_due_on_date(db, user.id, body.local_date)`. If habit_id not in [h.id for h in due_habits]: raise HTTPException(422, "Habit is not due on this date").
    5. Call `result = check_habit(db, user.id, habit_id, body.local_date)`.
    6. db.commit().
    7. Enrich response:
       - **Capsule enrichment:** If result["capsule"] is not None, query Reward by reward_id to get title and rarity. Build CapsuleDropDetail with id, reward_id, reward_title, reward_rarity.
       - **Quote selection:** Build a helper function `select_quote_for_context(db, result)` that picks the highest-priority trigger_event based on what happened:
         - Priority order: transformation (if transform_change is not None) > perfect_day (if is_perfect_day and is_checking) > zenkai (if zenkai_activated) > habit_complete (default when is_checking)
         - For unchecking (not is_checking): skip quote (return None)
         - Query Quote table filtered by trigger_event, order_by(func.random()).first()
         - Build QuoteDetail with avatar_path = f"/avatars/{quote.character}.png"
       - **Streak shaping:** Map result["streak"] to StreakInfo (current_streak, best_streak only).
       - **Habit streak shaping:** Map result["habit_streak"] to StreakInfo.
       - **Transform change:** Map result["transform_change"] to TransformChange or None.
       - **Dragon ball:** Map result["dragon_ball"] to DragonBallInfo or None.
    8. Return CheckHabitResponse.

    **Today's habits endpoint:**
    GET "/today/list":
    - Query param: local_date (str, required).
    - Call get_habits_due_on_date(db, user.id, local_date).
    - For each habit, query HabitLog (habit_id + log_date) for completion status, and HabitStreak for streak info.
    - Build HabitTodayResponse for each: all habit fields + completed (from HabitLog or False), streak_current, streak_best (from HabitStreak or 0/0).
    - Return list[HabitTodayResponse].

    **Calendar endpoint:**
    GET "/calendar/all":
    - Query param: month (str, required, format YYYY-MM).
    - Query DailyLog where user_id=user.id and log_date starts with month prefix (use LIKE 'YYYY-MM%').
    - Query OffDay where user_id=user.id and off_date starts with month prefix.
    - Build CalendarDay for each DailyLog: date, is_perfect_day, completion_tier, xp_earned, is_off_day (check if date in off_day_dates set).
    - Also include off days that have no DailyLog entry (pure off days): date, is_off_day=True, rest defaults.
    - Return list[CalendarDay] sorted by date.

    **Contribution graph endpoint:**
    GET "/{habit_id}/contribution-graph":
    - Query param: days (int, default=90).
    - Validate habit exists and belongs to user (404 if not).
    - Compute start_date = today - days (use the datetime module).
    - Query HabitLog where habit_id=habit_id and log_date >= start_date_str and completed=True.
    - Build set of completed dates.
    - Generate ContributionDay for each date in range: date (str), completed (bool).
    - Return list[ContributionDay].

    **Create `backend/app/api/v1/power.py`:**
    `router = APIRouter(prefix="/power", tags=["power"])`

    GET "/current":
    1. Read user's power_level, current_transformation, dragon_balls_collected, wishes_granted.
    2. Get current transformation info: `current_form = get_transformation_for_power(user.power_level)`.
    3. Find next transformation: iterate TRANSFORMATIONS, find the one after current_form. If current is the last, next is None.
    4. Build attributes array for each of VALID_ATTRIBUTES ("str", "vit", "int", "ki"):
       - raw_xp = getattr(user, f"{attr}_xp")
       - level = calc_attribute_level(raw_xp)
       - title = get_attribute_title(attr, level)
       - xp_for_next = get_xp_for_next_level(level)
       - To compute xp_for_current_level and progress_percent: iterate levels from 0, summing costs until reaching current level. xp_consumed = sum of all level costs up to current level. xp_into_current = raw_xp - xp_consumed. progress_percent = (xp_into_current / xp_for_next) * 100 if xp_for_next > 0 else 0.
       - Build AttributeDetail.
    5. Return PowerResponse.

    Add a second route at prefix="/attributes":
    GET "/attributes/" — Same attribute computation as above, return list[AttributeDetail]. Use a shared helper function `build_attribute_details(user)` to avoid duplication.

    `router = APIRouter(prefix="/analytics", tags=["analytics"])`

    GET "/summary":
    - Query param: period (Literal["week","month","all"], default="all").
    - Compute start_date based on period:
      - "week": today - 7 days
      - "month": today - 30 days
      - "all": no filter
    - Query DailyLog for user within date range.
    - Compute:
      - perfect_days = count where is_perfect_day=True

      - days_tracked = count of logs

    GET "/wish-history":
    - Query WishLog joined with Wish for user.
    - For each log: id, wish.title as wish_title, granted_at.
    - Return list[WishHistoryItem] ordered by granted_at desc.
