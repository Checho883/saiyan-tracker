1. Create `frontend/src/components/dashboard/ProgressRing.tsx`:

A reusable SVG circular progress ring component.

Props interface:
```typescript
interface ProgressRingProps {
  percentage: number;   // 0 to 1 (e.g., 0.85 = 85%)
  size?: number;        // px, default 72
  strokeWidth?: number; // px, default 5
  color: string;        // CSS color value, e.g. "var(--color-attr-str)"
  label: string;        // Display label below, e.g. "7-day" or "30-day"
}
```

Implementation:
- Calculate `radius = (size - strokeWidth) / 2` and `circumference = 2 * Math.PI * radius`
- Calculate `offset = circumference - (percentage * circumference)`
- Render an SVG with two circles:
  - Background circle: `fill="none"`, `stroke` using a muted color like `"var(--color-space-600)"`, full circumference
  - Progress circle: `fill="none"`, `stroke={color}`, `strokeDasharray={circumference}`, `strokeDashoffset={offset}`, `strokeLinecap="round"`
- Apply `className="-rotate-90"` on the SVG so progress starts from 12 o'clock (not 3 o'clock)
- Apply `transition-all duration-500 ease-out` on the progress circle for smooth animation
- Center the percentage text absolutely over the SVG: `{Math.round(percentage * 100)}%` in `text-text-primary font-bold text-sm`
- Below the SVG, show the label in `text-text-muted text-xs`
- Wrap in a flex-col items-center container
- Clamp percentage to 0-1 range: `const clamped = Math.max(0, Math.min(1, percentage))`
- Export as named export

interface CalendarGridProps {

Implementation:

- Component state: `currentMonth` (Date object, initialized to current month)
- Navigation: Previous/Next month buttons using ChevronLeft/ChevronRight icons
- Month header: display month name + year (e.g., "March 2026") with nav buttons on sides
- Build a dayMap from `days` array: `Map<string, HabitCalendarDay>` keyed by date string
- Day-of-week header row: S M T W T F S in `text-text-muted text-xs`
- Calculate first day of month's weekday to add empty padding cells
- Render each day of the month as a cell:

  - If not in data range: no dot
  - Today's date gets a subtle ring/border highlight
- Grid: CSS grid with 7 columns (`grid grid-cols-7 gap-1`)
- Export as named export
