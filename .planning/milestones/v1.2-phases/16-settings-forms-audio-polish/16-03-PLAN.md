<task type="auto">
  <name>Task 1: Generate individual sound effect WAV files using ffmpeg</name>
  <files>frontend/public/audio/</files>
  <action>
First, check that ffmpeg is available:
```bash
ffmpeg -version 2>&1 | head -1
```
If not available, install via: `choco install ffmpeg` or download from ffmpeg.org. If ffmpeg is unavailable, use a Node.js approach with the `audiosprite` npm package as fallback.

Create a temporary directory for individual sounds:
```bash
mkdir -p frontend/public/audio/tmp
```

Generate each sound using ffmpeg audio synthesis. Each is a distinct, punchy game-style sound:

**1. scouter_beep (400ms) — Electronic rising beep:**
```bash
ffmpeg -y -f lavfi -i "sine=frequency=800:duration=0.4" -af "afade=t=in:d=0.02,afade=t=out:d=0.1,asetrate=44100*1.5,aresample=44100,volume=0.7" frontend/public/audio/tmp/01_scouter_beep.wav
```

**2. power_up (1000ms) — Rising energy sweep:**
```bash
ffmpeg -y -f lavfi -i "sine=frequency=200:duration=1.0" -af "vibrato=f=8:d=0.5,afade=t=in:d=0.05,afade=t=out:d=0.3,asetrate=44100*2.0,aresample=44100,volume=0.6" frontend/public/audio/tmp/02_power_up.wav
```

**3. capsule_pop (300ms) — Quick pop:**
```bash
ffmpeg -y -f lavfi -i "anoisesrc=d=0.3:c=pink:a=0.5" -af "highpass=f=2000,afade=t=in:d=0.005,afade=t=out:d=0.15,volume=0.8" frontend/public/audio/tmp/03_capsule_pop.wav
```

**4. reveal_chime (500ms) — Bright bell chime:**
```bash
ffmpeg -y -f lavfi -i "sine=frequency=1200:duration=0.5" -af "tremolo=f=12:d=0.3,afade=t=in:d=0.01,afade=t=out:d=0.3,volume=0.6" frontend/public/audio/tmp/04_reveal_chime.wav
```

**5. explosion (1500ms) — Low boom:**
```bash
ffmpeg -y -f lavfi -i "anoisesrc=d=1.5:c=brown:a=0.8" -af "lowpass=f=400,afade=t=in:d=0.02,afade=t=out:d=0.8,volume=0.7" frontend/public/audio/tmp/05_explosion.wav
```

**6. radar_ping (600ms) — Sonar ping:**
```bash
ffmpeg -y -f lavfi -i "sine=frequency=1800:duration=0.6" -af "afade=t=in:d=0.01,afade=t=out:d=0.4,aecho=0.6:0.3:40:0.3,volume=0.5" frontend/public/audio/tmp/06_radar_ping.wav
```

**7. thunder_roar (2000ms) — Rolling thunder:**
```bash
ffmpeg -y -f lavfi -i "anoisesrc=d=2.0:c=brown:a=0.6" -af "lowpass=f=300,tremolo=f=3:d=0.7,afade=t=in:d=0.1,afade=t=out:d=1.0,volume=0.7" frontend/public/audio/tmp/07_thunder_roar.wav
```

**8. transform (2500ms) — Dramatic power surge:**
```bash
ffmpeg -y -f lavfi -i "sine=frequency=150:duration=2.5" -af "vibrato=f=6:d=0.8,afade=t=in:d=0.1,afade=t=out:d=0.5,asetrate=44100*1.8,aresample=44100,volume=0.7" frontend/public/audio/tmp/08_transform.wav
```

**9. swoosh (200ms) — Quick air movement:**
```bash
ffmpeg -y -f lavfi -i "anoisesrc=d=0.2:c=white:a=0.4" -af "highpass=f=1000,bandpass=f=3000:w=2000,afade=t=in:d=0.02,afade=t=out:d=0.1,volume=0.6" frontend/public/audio/tmp/09_swoosh.wav
```

**10. modal_open (150ms) — UI click up:**
```bash
ffmpeg -y -f lavfi -i "sine=frequency=600:duration=0.15" -af "afade=t=in:d=0.005,afade=t=out:d=0.08,asetrate=44100*1.3,aresample=44100,volume=0.5" frontend/public/audio/tmp/10_modal_open.wav
```

**11. modal_close (150ms) — UI click down:**
```bash
ffmpeg -y -f lavfi -i "sine=frequency=800:duration=0.15" -af "afade=t=in:d=0.005,afade=t=out:d=0.08,asetrate=44100*0.7,aresample=44100,volume=0.5" frontend/public/audio/tmp/11_modal_close.wav
```

**12. undo (250ms) — Descending two-tone:**
```bash
ffmpeg -y -f lavfi -i "sine=frequency=900:duration=0.25" -af "afade=t=in:d=0.01,afade=t=out:d=0.1,asetrate=44100*0.6,aresample=44100,volume=0.6" frontend/public/audio/tmp/12_undo.wav
```

**13. error_tone (200ms) — Dissonant buzz:**
```bash
ffmpeg -y -f lavfi -i "sine=frequency=250:duration=0.2" -af "tremolo=f=30:d=0.8,afade=t=in:d=0.01,afade=t=out:d=0.08,volume=0.7" frontend/public/audio/tmp/13_error_tone.wav
```

Verify all 13 files exist:
```bash
ls -la frontend/public/audio/tmp/*.wav | wc -l
```

<task type="auto">
  <name>Task 2: Compile sprite sheet and update SPRITE_MAP offsets</name>
  <files>frontend/public/audio/sprite.mp3, frontend/public/audio/sprite.webm, frontend/src/audio/soundMap.ts</files>
  <action>
1. **Create a concatenation list file** with 100ms silence gaps between sounds:

   First generate a 100ms silence file:
   ```bash
   ffmpeg -y -f lavfi -i "anullsrc=r=44100:cl=mono" -t 0.1 frontend/public/audio/tmp/silence.wav
   ```

   Create the concat list:
   ```bash
   cat > frontend/public/audio/tmp/concat.txt << 'EOF'
file '01_scouter_beep.wav'
file 'silence.wav'
file '02_power_up.wav'
file 'silence.wav'
file '03_capsule_pop.wav'
file 'silence.wav'
file '04_reveal_chime.wav'
file 'silence.wav'
file '05_explosion.wav'
file 'silence.wav'
file '06_radar_ping.wav'
file 'silence.wav'
file '07_thunder_roar.wav'
file 'silence.wav'
file '08_transform.wav'
file 'silence.wav'
file '09_swoosh.wav'
file 'silence.wav'
file '10_modal_open.wav'
file 'silence.wav'
file '11_modal_close.wav'
file 'silence.wav'
file '12_undo.wav'
file 'silence.wav'
file '13_error_tone.wav'
EOF
   ```

2. **Concatenate into a single WAV:**
   ```bash
   cd frontend/public/audio/tmp && ffmpeg -y -f concat -safe 0 -i concat.txt -c copy ../sprite_raw.wav
   ```

3. **Export as MP3 (192kbps):**
   ```bash
   ffmpeg -y -i frontend/public/audio/sprite_raw.wav -codec:a libmp3lame -b:a 192k frontend/public/audio/sprite.mp3
   ```

4. **Export as WebM (Opus codec):**
   ```bash
   ffmpeg -y -i frontend/public/audio/sprite_raw.wav -codec:a libopus -b:a 128k frontend/public/audio/sprite.webm
   ```

5. **Calculate offsets** by measuring each individual WAV duration:
   ```bash
   for f in frontend/public/audio/tmp/[0-9]*.wav; do
     name=$(basename "$f" .wav)
     duration=$(ffprobe -v error -show_entries format=duration -of csv=p=0 "$f" 2>/dev/null)
     echo "$name: ${duration}s"
   done
   ```

   Use the durations to calculate cumulative offsets (each sound starts after previous sound + 100ms gap). The executor should compute the actual millisecond values from ffprobe output and update SPRITE_MAP.

6. **Update frontend/src/audio/soundMap.ts** — Replace the SPRITE_MAP with computed offsets:

   The general pattern (executor must compute actual values from ffprobe output):
   ```typescript
   export const SPRITE_MAP: Record<SoundId, [number, number]> = {
     scouter_beep: [0, <actual_duration_ms>],
     power_up: [<offset>, <actual_duration_ms>],
     capsule_pop: [<offset>, <actual_duration_ms>],
     reveal_chime: [<offset>, <actual_duration_ms>],
     explosion: [<offset>, <actual_duration_ms>],
     radar_ping: [<offset>, <actual_duration_ms>],
     thunder_roar: [<offset>, <actual_duration_ms>],
     transform: [<offset>, <actual_duration_ms>],
     swoosh: [<offset>, <actual_duration_ms>],
     modal_open: [<offset>, <actual_duration_ms>],
     modal_close: [<offset>, <actual_duration_ms>],
     undo: [<offset>, <actual_duration_ms>],
     error_tone: [<offset>, <actual_duration_ms>],
   };
   ```

   **Computing offsets:** offset_N = offset_(N-1) + duration_(N-1) + 100 (gap). Start at offset 0 for scouter_beep. Round all values to nearest integer millisecond.

7. **Clean up temporary files:**

   ```bash
   ffprobe -v error -show_entries format=duration -of csv=p=0 frontend/public/audio/sprite.mp3
   ```
   Both files should exist with non-zero size. Total duration should be roughly 9-10 seconds.
