    **check_zenkai_recovery(db: Session, user_id: UUID, local_date: str, streak: Streak) -> dict**
    - If streak.last_active_date is None or streak.current_streak == 0: return {zenkai_activated: False, halved_from: 0, new_streak: 0}
    - Parse dates, compute gap_days = (today - last_date).days
    - If gap_days <= 1: return {zenkai_activated: False} (no gap)
    - Count off days in range (exclusive of both endpoints): query OffDay where off_date > last_active_date AND off_date < local_date
    - If off_day_count >= gap_days - 1: streak continues unbroken
    - Else: streak break detected. `new_streak = max(streak.current_streak // 2, 0)`. Return {zenkai_activated: True, halved_from: old, new_streak: new}
    - CRITICAL: Do NOT mutate streak here — the caller (check_habit) decides when to apply.

    **update_overall_streak(db: Session, user_id: UUID, local_date: str, completion_rate: float, zenkai_info: dict) -> dict**
    - Get or create streak.
    - If zenkai_info["zenkai_activated"]: set streak.current_streak = zenkai_info["new_streak"]
    - If completion_rate >= STREAK_MIN_COMPLETION:
      - streak.current_streak += 1
      - streak.best_streak = max(streak.best_streak, streak.current_streak)
    - Else: streak does not increment (but does not reset — only Zenkai halves, and that's already applied above)
    - streak.last_active_date = local_date
    - Return {"current_streak": streak.current_streak, "best_streak": streak.best_streak, "zenkai_activated": zenkai_info.get("zenkai_activated", False), "halved_from": zenkai_info.get("halved_from", 0)}

    - Sets HabitLog.completed = False, claws back attribute_xp_awarded
    - Claws back attribute XP from user
    - Recalculates DailyLog (completion rate drops, tier may change)
    - Recalculates daily XP
    - Resets habit streak to 0
    - Recalculates power level
    - Revokes Dragon Ball if was Perfect Day and no longer is
    - Does NOT remove capsule drop (capsules stick)
    - Returns updated response dict

    check_habit() — re-checking after uncheck:
    - Re-awards attribute XP
    - Does NOT re-roll capsule (capsule_dropped flag already True)

    get_habits_due_on_date:
    - Returns daily habits every day
    - Returns weekday habits Mon-Fri only
    - Returns custom habits on matching custom_days
    - Excludes inactive habits
    - Excludes habits not started yet or already ended

  <action>
    1. **Create `backend/app/services/habit_service.py`:**

    **get_habits_due_on_date(db: Session, user_id: UUID, local_date: str) -> list[Habit]**
    - Parse local_date to date object.
    - Query active habits for user where start_date <= local_date and (end_date is None or end_date >= local_date).
    - Filter by frequency:
      - "daily": always due
      - "weekdays": due if date.weekday() < 5 (Mon=0 through Fri=4)
      - "custom": due if date.isoweekday() is in habit.custom_days list (custom_days stores ISO weekday numbers 1-7)
    - Return list of matching Habit objects.

    **check_habit(db: Session, user_id: UUID, habit_id: UUID, local_date: str) -> dict**
    Implement as a single atomic transaction with try/except and rollback:

    Step 1 — Toggle habit log:
    - Query HabitLog by habit_id + log_date.
    - If not exists: create with completed=True → is_checking = True
    - If exists and completed=True: set completed=False, clear completed_at → is_checking = False
    - If exists and completed=False: set completed=True, set completed_at → is_checking = True

    Step 2 — Handle attribute XP:
    - Get habit from DB (need importance + attribute).
    - If is_checking: `xp = get_attribute_xp(habit.importance)`, set `habit_log.attribute_xp_awarded = xp`, add XP to `user.{habit.attribute}_xp`
    - If unchecking: claw back `habit_log.attribute_xp_awarded` from `user.{habit.attribute}_xp`, set `habit_log.attribute_xp_awarded = 0`

    Step 3 — Update daily log:
    - Get all habits due today: `habits_due = get_habits_due_on_date(db, user_id, local_date)`
    - Count completed: query HabitLog where habit_id in due_habit_ids, log_date = local_date, completed = True
    - Compute completion_rate = completed_count / len(habits_due) (guard: if 0 habits due, rate = 0.0)
    - Get tier = get_completion_tier(completion_rate)
    - Get or create DailyLog for user_id + log_date
    - Update: habits_due, habits_completed, habit_completion_rate, is_perfect_day (rate == 1.0), completion_tier

    Step 4 — Check Zenkai recovery (only on first check of the day):
    - Get or create Streak for user.
    - If this is the first HabitLog entry for this date (i.e., was_new_log): call check_zenkai_recovery()
    - Set daily_log.zenkai_bonus_applied if zenkai activated

    Step 5 — Update streaks:
    - Call update_overall_streak() with completion_rate and zenkai_info
    - Call update_habit_streak() with is_checking

    Step 6 — Recalculate daily XP:
    - `daily_xp = calc_daily_xp(completion_rate, streak.current_streak, daily_log.zenkai_bonus_applied)`
    - `daily_log.xp_earned = daily_xp`
    - `daily_log.streak_multiplier = 1 + calc_streak_bonus(streak.current_streak)`

    Step 7 — Update power level and transformation:
    - `old_transformation = user.current_transformation`
    - `new_power = recalculate_power_level(db, user)`
    - `transform_change = check_transformation_change(old_transformation, new_power)`
    - If transform_change: `user.current_transformation = transform_change["key"]`

    Step 8 — Handle Dragon Ball:
    - If is_perfect_day and not daily_log.dragon_ball_earned:
      - `db_info = award_dragon_ball(user)`
      - `daily_log.dragon_ball_earned = True`
    - If was_perfect_day and not is_perfect_day (unchecking broke perfection):
      - `revoke_dragon_ball(user)`
      - `daily_log.dragon_ball_earned = False`

    Step 9 — Capsule RNG:
    - Only if is_checking and not habit_log.capsule_dropped:
      - `capsule = roll_capsule_drop(db, user_id, habit_id)`
      - If capsule: `db.add(capsule)`, `habit_log.capsule_dropped = True`

    Step 10 — Commit:
    - `db.commit()`
    - Build and return response dict with all fields per GAME-14 / API-01 spec.

    Wrap entire function in try/except: on any exception, `db.rollback()` then `raise`.
