   Visual design phases (phased reveal like TransformationOverlay):
   - Phase 0 (0ms): Initial flash — bright cyan/blue flash covering screen, fades in 400ms
   - Phase 1 (300ms): Gradient background — radial gradient from-cyan-600/80 to-blue-900/40
   - Phase 2 (600ms): Center text "ZENKAI BOOST!" in large bold white with cyan text-shadow glow, spring scale-in (SPRINGS.bouncy)
   - Phase 3 (900ms): Subtitle "A Saiyan grows stronger after every defeat!" fade-in
   - Phase 4 (1200ms): "Welcome back, warrior" in smaller text-white/70, fade-in
   - Auto-complete after 3 seconds
   - useSkippable with 1000ms guard (dramatic moment, longer guard)
   - data-testid="zenkai-recovery-overlay"

   Use phased useState pattern from TransformationOverlay:
   ```typescript
   const [phase, setPhase] = useState(0);
   useEffect(() => {
     const timers = [
       setTimeout(() => setPhase(1), 300),
       setTimeout(() => setPhase(2), 600),
       setTimeout(() => setPhase(3), 900),
       setTimeout(() => setPhase(4), 1200),
       setTimeout(() => onComplete(), 3000),
     ];
     return () => timers.forEach(clearTimeout);
   }, [onComplete]);
   ```
