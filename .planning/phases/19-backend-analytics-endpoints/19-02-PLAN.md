    - Test: halved_value = old_streak // 2

  <action>
1. Add schemas to `backend/app/schemas/status.py`:
   ```python
   class StreakBreak(BaseModel):
       habit_id: str
       habit_title: str
       old_streak: int
       halved_value: int

   class StatusResponse(BaseModel):
       welcome_back: StatusQuote | None = None
       roast: RoastInfo | None = None
       streak_breaks: list[StreakBreak] = []  # NEW — default empty for backward compat
   ```

2. Add streak-break detection logic to `status.py`:
   - Import HabitStreak, Habit, OffDay models
   - After getting welcome_status from roast_service, detect per-habit streak breaks:
     a. Query all HabitStreaks for user where current_streak > 0 and last_completed_date is not None
     b. Join with Habit to get title
     c. For each, compare last_completed_date to local_date:
        - Calculate gap_days = (today - last_completed).days
        - If gap_days <= 1: no break, skip
        - Count off-days in the gap (OffDay records between last_completed_date and local_date, exclusive)
        - If off_day_count >= gap_days - 1: all gap days are off-days, no break
        - Otherwise: streak break detected. old_streak = current_streak, halved_value = current_streak // 2
   - This is stateless — no mutation of HabitStreak. Detection only.
   - Add streak_breaks list to StatusResponse return

   - Test no streak breaks when no habits have streaks
   - Test streak break detected: create HabitStreak with current_streak=10, last_completed_date=3 days ago, no off-days
   - Test off-days prevent break: create off-days covering the gap
   - Test response shape: each entry has habit_id, habit_title, old_streak, halved_value
   - Test halved_value calculation: 10 → 5, 7 → 3, 1 → 0
