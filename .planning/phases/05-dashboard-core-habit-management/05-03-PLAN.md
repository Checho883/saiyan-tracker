Add imports for `HabitCreate`, `HabitUpdate` from types.

Implement following rewardStore CRUD pattern:
- `createHabit`: call `habitsApi.create(data)`, then re-fetch today's habits with `get().fetchToday(today)` to get the updated list (don't append locally since todayList filters by date/frequency)

- Today's date: `new Date().toISOString().split('T')[0]`
