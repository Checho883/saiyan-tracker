    Create one file per model. Follow PRD Section 8 schemas EXACTLY for column names, types, and constraints. Use SQLAlchemy 2.0 patterns from research (Mapped[], mapped_column, DeclarativeBase).

    **Critical rules for ALL models:**
    - Import Base from app.database.base
    - Use uuid.UUID type with sqlalchemy.Uuid for all primary keys (default=uuid.uuid4)
    - Use String references in relationship() and ForeignKey() to avoid circular imports
    - Date-logic columns (log_date, off_date, start_date, end_date, last_completed_date, last_active_date) use Mapped[str] = mapped_column(String(10)) for YYYY-MM-DD
    - Metadata timestamps (created_at, completed_at, dropped_at, unlocked_at, granted_at) use Mapped[datetime] with default=datetime.utcnow
    - Use Optional[] and nullable=True for nullable columns

    **Per-model specifics (follow PRD Section 8 exactly):**

    **User (users):** id, username(String(100) default="default-user"), display_name(String(100) default="Saiyan"), dragon_balls_collected(int default=0), wishes_granted(int default=0), str_xp(int default=0), vit_xp(int default=0), int_xp(int default=0), ki_xp(int default=0), power_level(int default=0), current_transformation(String(20) default="base"), sound_enabled(bool default=True), theme(String(10) default="dark"), created_at. Relationships: habits, categories, habit_logs, daily_logs, streaks, rewards, wishes, off_days, achievements.

    **Category (categories):** id, user_id(FK users.id), name(String(100)), color_code(String(7)), icon(String(10)), sort_order(int default=0), created_at. NO point_multiplier. Relationships: user, habits.

    **Habit (habits):** id, user_id(FK users.id), category_id(FK categories.id, ondelete="SET NULL", nullable=True), title(String(255)), description(Text nullable), icon_emoji(String(10) default="⭐"), importance(String(10) default="normal"), attribute(String(3)), frequency(String(20) default="daily"), custom_days(JSON nullable), target_time(String(10) nullable), is_temporary(bool default=False), start_date(String(10)), end_date(String(10) nullable), sort_order(int default=0), is_active(bool default=True), created_at. Relationships: user, category, logs, habit_streak.

    **HabitLog (habit_logs):** __table_args__ with UniqueConstraint("habit_id", "log_date", name="uq_habit_log_date"). id, user_id(FK users.id), habit_id(FK habits.id), log_date(String(10)), completed(bool default=False), completed_at(datetime nullable), attribute_xp_awarded(int default=0), capsule_dropped(bool default=False), notes(Text nullable). Relationships: user, habit.

    **HabitStreak (habit_streaks):** id, user_id(FK users.id), habit_id(FK habits.id), current_streak(int default=0), best_streak(int default=0), last_completed_date(String(10) nullable). Relationships: user, habit.

    **DailyLog (daily_logs):** __table_args__ with UniqueConstraint("user_id", "log_date", name="uq_user_log_date"). id, user_id(FK users.id), log_date(String(10)), habits_due(int default=0), habits_completed(int default=0), habit_completion_rate(Float default=0.0), is_perfect_day(bool default=False), completion_tier(String(20) default="base"), xp_earned(int default=0), streak_multiplier(Float default=1.0), zenkai_bonus_applied(bool default=False), dragon_ball_earned(bool default=False). Relationships: user.

    **Streak (streaks):** id, user_id(FK users.id), current_streak(int default=0), best_streak(int default=0), last_active_date(String(10) nullable). Relationships: user.

    **PowerLevel (power_levels):** id, user_id(FK users.id), log_date(String(10)), total_points(int default=0), transformation_level(String(20) default="base"). Relationships: user.

    **Reward (rewards):** id, user_id(FK users.id), title(String(255)), rarity(String(20) default="common"), is_active(bool default=True), created_at. Relationships: user, capsule_drops.

    **CapsuleDrop (capsule_drops):** id, user_id(FK users.id), reward_id(FK rewards.id), habit_id(FK habits.id), dropped_at(datetime default=datetime.utcnow), claimed(bool default=False). Relationships: user, reward, habit.

    **Wish (wishes):** id, user_id(FK users.id), title(String(255)), is_active(bool default=True), times_wished(int default=0), created_at. Relationships: user, wish_logs.

    **WishLog (wish_logs):** id, user_id(FK users.id), wish_id(FK wishes.id, ondelete="CASCADE"), granted_at(datetime default=datetime.utcnow). Relationships: user, wish.

    **OffDay (off_days):** id, user_id(FK users.id), off_date(String(10)), reason(String(20) default="rest"), notes(Text nullable), created_at. Relationships: user.
