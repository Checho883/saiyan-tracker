Create `backend/app/services/roast_service.py`:

1. `calc_roast_severity(effective_gap: int) -> str | None`
   - Pure function
   - If effective_gap <= 0: return None
   - If effective_gap <= 2: return "mild"
   - If effective_gap <= 6: return "medium"
   - If effective_gap >= 7: return "savage"

2. `detect_absence_gap(db: Session, user_id: UUID, today_str: str) -> dict`
   - Get streak via get_or_create_streak(db, user_id)
   - If streak.last_active_date is None, return {"gap_days": 0, "off_days_in_gap": 0, "effective_gap": 0, "severity": None}
   - Calculate gap_days = (today - last_active_date).days
   - If gap_days <= 1, return {"gap_days": 0, "off_days_in_gap": 0, "effective_gap": 0, "severity": None}
   - Count off-days in the gap (same pattern as check_zenkai_recovery):
    ```python
    off_day_count = db.query(OffDay).filter(
        OffDay.user_id == user_id,
        OffDay.off_date > streak.last_active_date,
        OffDay.off_date < today_str,
    ).count()
    ```
   - effective_gap = gap_days - 1 - off_day_count (subtract 1 because gap_days includes the day after last active)
   - severity = calc_roast_severity(effective_gap)
   - Return dict with all values

3. `get_welcome_status(db: Session, user_id: UUID, today_str: str) -> dict`
   - Call detect_absence_gap(db, user_id, today_str)
   - If severity is None: return {"welcome_back": None, "roast": None}
   - Fetch random Goku welcome_back quote: `db.query(Quote).filter(Quote.trigger_event == "welcome_back", Quote.character == "goku").order_by(func.random()).first()`
   - Fetch random Vegeta roast quote with matching severity: `db.query(Quote).filter(Quote.trigger_event == "roast", Quote.character == "vegeta", Quote.severity == severity).order_by(func.random()).first()`
   - Build response with quote dicts (character, quote_text, source_saga, avatar_path)
   - Return {"welcome_back": welcome_quote_dict, "roast": {"quote": roast_quote_dict, "severity": severity, "gap_days": effective_gap}}

**Create `backend/app/schemas/status.py`:**
```python
from pydantic import BaseModel

class StatusQuote(BaseModel):
    character: str
    quote_text: str
    source_saga: str
    avatar_path: str

class RoastInfo(BaseModel):
    quote: StatusQuote | None = None
    severity: str
    gap_days: int

class StatusResponse(BaseModel):
    welcome_back: StatusQuote | None = None
    roast: RoastInfo | None = None
```

**Create `backend/app/api/v1/status.py`:**
```python
from fastapi import APIRouter, Depends, Query
from sqlalchemy.orm import Session

from app.api.deps import get_current_user, get_db
from app.models.user import User
from app.schemas.status import StatusResponse
from app.services.roast_service import get_welcome_status

router = APIRouter(prefix="/status", tags=["status"])

@router.get("/", response_model=StatusResponse)
def get_status(
    local_date: str = Query(..., pattern=r"^\d{4}-\d{2}-\d{2}$"),
    db: Session = Depends(get_db),
    user: User = Depends(get_current_user),
):
    result = get_welcome_status(db, user.id, local_date)
    return StatusResponse(**result)
```
