**After user downloads are complete**, Claude will:

1. **Sanitize any SVG sources** (ART-15): For each `.svg` file in `raw/`:
   - Check for `<script>`, `on*=` event handlers, `javascript:` URLs, `xlink:href` to external URLs, `<foreignObject>`, `<iframe>`, `<embed>`, `url(http`, `url(data:text/html`
   - Remove any dangerous elements/attributes found
   - Log sanitization results

2. **Install sharp** as a dev dependency: `cd frontend && npm install --save-dev sharp`

3. **Create processing script** at `frontend/scripts/process-avatars.mjs`:
   ```javascript
   import sharp from 'sharp';
   import { readdirSync, existsSync, mkdirSync } from 'fs';
   import { join, basename, extname } from 'path';

   const RAW_DIR = 'public/assets/raw';
   const OUT_DIR = 'public/assets/avatars';

   // Map raw filenames to output names
   const FORM_MAP = {
     'base-goku': 'base',
     'ssj-goku': 'ssj',
     'ssj2-goku': 'ssj2',
     'ssj3-goku': 'ssj3',
     'ssg-goku': 'ssg',
     'ssb-goku': 'ssb',
     'ui-goku': 'ui',
     'goku-portrait': 'goku',
     'vegeta-portrait': 'vegeta',
   };

   // Avatar dimensions (2x retina)
   const AVATAR_SIZE = 256;   // Transformation avatars (displays at 96px)
   const PORTRAIT_SIZE = 128; // Character portraits (displays at 40px)

   if (!existsSync(OUT_DIR)) mkdirSync(OUT_DIR, { recursive: true });

   const files = readdirSync(RAW_DIR).filter(f => !f.startsWith('.'));

   for (const file of files) {
     const name = basename(file, extname(file));
     const outName = FORM_MAP[name];
     if (!outName) {
       console.log(`Skipping unknown file: ${file}`);
       continue;
     }

     const isPortrait = outName === 'goku' || outName === 'vegeta';
     const size = isPortrait ? PORTRAIT_SIZE : AVATAR_SIZE;

     try {
       await sharp(join(RAW_DIR, file))
         .resize(size, size, { fit: 'cover', position: 'top' })
         .webp({ quality: 85, alphaQuality: 100 })
         .toFile(join(OUT_DIR, `${outName}.webp`));
       console.log(`✓ ${file} → ${outName}.webp (${size}x${size})`);
     } catch (err) {
       console.error(`✗ ${file}: ${err.message}`);
     }
   }
   ```

4. **Run the script**: `cd frontend && node scripts/process-avatars.mjs`

5. **Verify output**: Check that all 9 WebP files exist and have reasonable file sizes (each should be 5KB-200KB).

6. **Clean up**: Remove `sharp` dev dependency if not needed long-term: `cd frontend && npm uninstall sharp`. Delete `frontend/scripts/process-avatars.mjs`. The processing is a one-time operation.
